# 月度账单快照

一个纯前端的月度资产快照与闭环排查工具。

## 数据与隐私

- 所有输入仅保存在当前浏览器的 `localStorage`。
- 不上传账户数据，也不包含原作者的私人账单或 Excel 文件。
- 不同设备和浏览器之间不会同步。
- 清除浏览器站点数据会删除本地记录。

## 使用

1. 第一次填写账户余额后保存初始快照。
2. 下一次盘点时填写新余额、期间收入和大开销。
3. 查看基本日均、体感差异和排查助手。
4. 确认后选择“保存并开始下月”，结果会进入当前设备的历史库。

## GitHub Pages

仓库设置中选择 **Pages → Deploy from a branch → main / root** 即可发布。
